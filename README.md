# AppClasse

Ce projet Java permet de gérer une classe d'élèves.

## Fichiers principaux
- `AppliClasseEleves.java` : Point d'entrée de l'application
- `Classe.java` : Gestion de la classe et des élèves
- `Eleve.java` : Représentation d'un élève

## Utilisation
Compilez et exécutez `AppliClasseEleves.java` pour lancer l'application.
